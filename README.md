# Planting Hope in a Warming World

A blog post about Simeon Kalua's climate justice work in Malawi.

This blog was written for high school students as part of a coursework assignment exploring climate change, health, and leadership. It highlights the work of Simeon Kalua, a Malawian climate justice activist and permaculturist working at the intersection of planetary health and human wellbeing.

## About the Blog

The blog explores how Simeon Kalua is making a difference through community-led climate action in Malawi. Topics covered include:

- The real, lived impacts of climate change on farming communities
- How permaculture and sustainable land-use build resilience
- The deep connection between climate change and human health
- Why climate leadership can be local, practical, and deeply human
- The importance of inclusive climate justice

## Features

- Scroll-reveal animations on each section

- Ambient nature sound toggle (Web Audio API)
- Animated leaf particles on the hero section
- Embedded photographs from Simeon's advocacy work
- Responsive design for mobile and desktop
- Numbered references with linked sources

## How to Use

1. Clone or download this repository
2. The entire blog is contained in a single `index.html` file (all images are embedded)
3. Open `index.html` in any browser, or deploy via GitHub Pages

### Deploy with GitHub Pages

1. Go to **Settings** then **Pages**
2. Under Source, select **Deploy from a branch**
3. Choose **main** branch and click **Save**
4. Your site will be live within a few minutes
